Login Screen

1. Screen title

2. Enter username

3. Enter password

4. Initiates login process – if successful, takes user to homepage

5. If user is new and wants to register

In order to access Holidaze's capabilities, you must login using your username and password. If you do not have an account, press the register button to sign up on another screen. Once your email and password have been correctly entered and the login button is pressed, you will be taken to the homepage, from which you can use the app's functionalities.

Pressing the back button on your phone/tablet will take you out of the app.



Sign-up Screen

1. Screen title

2. User’s full name

3. User’s age

4. User’s email

5. User’s password

6. User is registered with above details

To register, you must enter your full name, age (as an integer), a valid email that you have access to, and a password of at least 6 characters. Once these are entered properly, you can press the register user button, which will send you an email prompting you to verfiy your email. After verifying, you can press the back button to go to the login page again, where your email and password will grant access to the app's functionalities. This is powered by Firebase.

Pressing the back button on your phone/tablet will take you to the login screen.



Homepage

1. Screen title

2. Takes user to the user guide

3. Takes user to the bucket list page

4. Takes user to search page

5. Logs user out (on long press)

Each button will take you to their respective screens, and you may only log out by holding down the darker blue log out button.

Pressing the back button on your phone/tablet will take you to the login screen.



Search Screen

1. Enter origin airport IATA code (3-letters)

2. Enter destination airport IATA code (3-letters)

3. Departure date picker

4. Return date picker

5. Begin search for cheapest direct flight

This is the most important aspect of Holidaze. In order to get results, you must enter valid IATA codes for the respective airports (not case-sensitive). For example, entering "heathrow" or "dubai" will not work. You must select a date within 6 months of the current date. If there are no direct flights available on your chosen dates, you will be notified. Powered by Expedia. A sound will be played when the search button is pressed, stopping when results are found.

Pressing the back button on your phone/tablet will take you to the homepage.



Search Results

1. Displays total price in USD

2. Displays airline, flight number, flight time for departure leg

3. Displays airline, flight number, flight time for return leg

This page displays the cheapest direct flights on the given dates, as well as the total price for the return trip in USD, the flight times, flight numbers, and airlines.

Pressing the back button on your phone/tablet will take you to the search screen.



Bucket List

1. Adds a new item to bucket list

2. Lists current items (scrollable)

Here, you can see all of the items in your bucket list. Unfortunately, if you exit to the homepage and return to the bucket list screen, the items all appeared to be added again. However, rest assured that the content is not permanent, and if you were to close the app and return to the bucket list, it would contain the correct content. Powered by SQLite

Pressing the back button on your phone/tablet will take you to the homepage.



Adding/Editing bucket list items

1. Save new item/ update old one

2. Item title (e.g. Dubai – DXB trip)

3. Item description (e.g. May 19th - June 1st)

4. Item deletion

This acitivity is accessed when a new item is created, or pressed on (both from the bucket list screen) allows you to edit or delete the item. Pressing save or delete will return you to the bucket list page, with the chosen command executed.

Pressing the back button on your phone/tablet will take you to your bucket list
